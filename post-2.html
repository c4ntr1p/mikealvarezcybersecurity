Scope of Capstone

This paper will follow the thinking points presented by my thesis statement in the following ways:
  • Discuss what Application Security is.
  • Describe common web application vulnerabilities, exploits, and what causes them.
  • How to identify malware in web applications
  • Various coding languages and how they help in web security
  • What companies are doing to make apps safe
  • What freelancers are doing to help where enterprise companies cannot
Topics to Consider:
  • How is application security distinct? Do we need to be so granular within cybersecurity?
  • Dangers of XSS attacks; what causes them, and what they can do?
  • Dangers of SQL Injection; what causes them, and what can they do?
  • Bug Bounties, Burp Suite, and how to find vulnerabilities; metrics to show bug bounty programs work and make the environment safer.
